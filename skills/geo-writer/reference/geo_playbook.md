# Playbook GEO — comment se faire citer par les LLM

Tiré des cas Lysol / Accor (deck MentionLab) et des diagnostics sources réels.

## Pourquoi
La recherche bascule vers une réponse générée unique. Pas **mentionné** = disparu au moment du choix ; pas **cité** = pas de contrôle du narratif. Le contenu GEO vise les deux.

## Les 7 leviers
1. **Q&A, réponse d'abord** : question réelle en H2, réponse 1–3 phrases dessous, détail ensuite. Les pages produit marketing donnent peu d'info exploitable.
2. **Structurer pour la machine** : tableaux, listes, H2/H3 explicites, phrases courtes autonomes. Ce qui est scannable est citable.
3. **Matcher la langue du marché** : les LLM préfèrent la langue de la question (Lysol −42% de mentions en espagnol sans site ES).
4. **Volume de pages pertinentes** : Clorox 3× plus de pages → 3× plus de citations.
5. **Être présent là où le LLM puise** : Reddit, YouTube, forums, sites d'autorité (Accor : 81% citations Gemini = YouTube ; 27% AI Overview = Facebook).
6. **Fraîcheur datée** : année en cours, « mis à jour le… » → déclenche la recherche live.
7. **Clarté entité + JSON-LD** : nommer marque/produits/chiffres ; FAQPage/Article.

## Diagnostic → angle
| Faiblesse | Lecture | Type d'article |
|---|---|---|
| Visibilité basse + ranking bon | Couverture | Q&A informatif + plus de pages |
| Visibilité basse + ranking mauvais | Position/autorité | Comparatif + validation tierce |
| Sentiment bas (Brand) | Narratif négatif | Message de marque, FAQ, preuves |
| Source = forum/YouTube/Reddit | Hors owned | Présence/format adapté |

## À ne pas faire
Pas d'intro marketing avant la réponse ; pas de prose là où un tableau suffit ; pas de chiffres inventés/non datés ; pas un article fourre-tout (une intention par page).
