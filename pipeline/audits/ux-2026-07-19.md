### 2. meilleur-abonnement-5g.be — Direction DA #4 « Premium sombre », mutée

**DA / Direction ⚠️**
- Direction assumée et unique : DA-DIRECTIONS #4 « Premium sombre » mutée (périwinkle→turquoise `#4FD1C5`, graphite bleuté froid), mode **DARK fixe** (`data-theme="dark"` + `da-site.css` en dernier, `!important` neutralisant les 3 contextes), fonts `Space Grotesk` + `IBM Plex Sans`. Palette mutée et propre.
- **Risque token important** : `app/globals.css` conserve la palette **TEMPLATE dans le bloc Tailwind `@theme`** (`--color-accent-1: #FF3D57` rouge, ambre, etc.). `da-site.css` n'override que les variables `:root` (`--accent-*`), **pas les tokens `@theme`** qui pilotent la génération des utilitaires Tailwind v4. Tout composant utilisant une classe utilitaire `*-accent-*` (ex. `bg-accent-1`, `text-accent-2`) rendrait donc le **rouge/ambre du template, hors-DA**. À synchroniser (contraste avec le site énergie qui, lui, a réécrit `@theme`).
- `globals.css` garde aussi les blocs complets `prefers-color-scheme: light` + `html[data-theme=...]` (neutralisés par `da-site.css !important`) — fonctionne mais fragile/lourd.

**Logo / Identité ⚠️**
- Logo Nav = **mark SVG inline** (3 arcs de diffusion + base pleine, « le signal qui part d'une antenne »), tinté `var(--accent-1)` via `da-site.css`. Favicon `app/icon.svg` = **le même mark** → cohérence favicon↔logo ✅ (mieux que le site énergie).
- ⚠️ Motif **proche d'un glyphe « réseau/diffusion » générique** (arcs de signal ~ icône wifi standard). Thématiquement juste pour la 5G mais à surveiller côté originalité — ce n'est pas l'« éclair » partagé du réseau, mais ça reste un symbole de connectivité attendu.
- OG : image dynamique aux couleurs de la palette ; accroche honnête (« Classement · Comparateur · Guides — sans affiliation », omet quiz/simulateur/deals désactivés). ✅

**Images ⚠️**
- Featured OK : article `proximus-orange-telenet-quel-reseau-5g` a `featureImage: "/images/blog/a5g-seed-cover.jpeg"` + `alt`. ✅
- **Covers de catégorie non rendues sur la page catégorie** : fichiers présents (`public/images/categories/a5g-cat-*.jpeg`) mais même `CategoryView` sans image de hero → covers en `public/` seulement.
- Images in-body = reprise des covers catégorie/blogcat (`a5g-cat-operateurs.jpeg`, `a5g-blogcat-couverture-w.jpeg`) — non uniques à l'article.
- Positif : `alt` descriptifs, table markdown des débits présente (wrappée via override MDX `.table-scroll-wrap`).

**Responsive ✅**
- `da-site.css` : `html, body { max-width:100%; overflow-x:hidden }` ; garde-fous globals ; menu mobile (`@940px`) ; table wrappée. RAS.

**A11y ✅**
- `lang="en"` via wrapper `/en` (`display:contents`). Skip-link, `:focus-visible`, `prefers-reduced-motion`. Ordre des titres cohérent.
