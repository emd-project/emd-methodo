## ⏭ Reporté au prochain run (avec raison)

- **A11y — `<html lang>` dynamique en SSR** (utilitaire, 7-places, neobanque + autres bilingues à `lang="fr"` figé). **Reporté :** nécessite une infra `getLocale()` serveur (cookie/header/middleware). Seul `meilleure-voiture.be` la possède (`@/lib/i18n/server-locale`) ; `utilitaire/lib/i18n` ne contient que `article-slugs.ts`. Créer une détection de locale SSR à l'aveugle = risque de **build cassé** (import d'un module inexistant) → contraire à « code compilable ». À traiter avec un audit dédié de l'infra i18n (reconstruction, hors périmètre image/responsive de ce run).
- **meilleure-voiture-7-places.be — radius soft vs `shape:'sharp'`.** `globals.css` porte des rayons doux (`--radius-* 6→28px`) alors que `permutations.shape='sharp'`. **Reporté :** incohérence à résoudre côté composant `PermutationStyle` (qui doit imposer les rayons selon `shape`), pas par un override CSS large qui risquerait de dégrader tous les composants. Cosmétique (⚠️, non bloquant).
- **Jpeg orphelins à la racine** (familiale, utilitaire). **Reporté :** le MCP GitHub disponible (`github_write_file`/`github_commit_batch`) **n'expose pas de suppression de fichier**. Assets non référencés (hygiène ⚠️ faible), aucun impact rendu. À nettoyer via un outil de delete.
- **meilleure-voiture-electrique — covers d'article non uniques** (1 cover partagée). **Reporté :** la branche par défaut du repo n'est **pas `main`** (`claude/no-image-spec-generator-…`) ; générer/committer sur `main` demande de vérifier l'état réel de `main` d'abord (hors budget de ce run). Génération de covers 16:9 à programmer.
- **`globals.css :root` néon inerte** (voiture.be, familiale, utilitaire, quel-operateur, énergie, neobanque). Les littéraux template subsistent dans `globals.css` mais sont **systématiquement surchargés** par la couche DA chargée en dernier (`volteo-overrides.css` / `brand-tokens.css` / `da-site.css`) → **non rendus**. Dette template récurrente : à traiter **à la source du template** (`emd-template`), pas par churn per-site.
